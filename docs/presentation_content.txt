SLIDE 2 — THE PROBLEM
----------------------
The problem engineers face today:

- GM Bolt recall 2021 — 141,667 vehicles, $1.9B cost — thermal runaway
- China's Dec 2023 graphite export ban — 77% of global supply, no warning
- DRC cobalt — 73% concentration, FEOC non-compliant under US IRA 2022

Manual analysis of these risks takes 3–6 weeks per BOM revision.
There is no agentic tool at the engineering-decision layer.


SLIDE 3 — THE SOLUTION
-----------------------
Upload a Bill of Materials JSON → get a full risk report in 90 seconds

Four things MatRes tells you:
1. Which materials are geopolitically concentrated (HHI score + FEOC flag)
2. What failures those materials have caused in real vehicles (NHTSA recalls)
3. What you can substitute them with and how the properties compare
4. How long and how expensive the qualification process will be

Every number has a source citation. No uncited claims reach the UI.

SLIDE 7 — THE HALLUCINATION GUARD
-----------------------------------
Why this matters for engineering decisions:

Engineers cannot act on uncited AI output — legal and regulatory exposure.

MatRes enforces a citation rule at the middleware layer:
Every numeric value must carry a source field before reaching the UI.
If a sub-agent returns a number without a citation → CitationError is raised.

Example:
  "Cobalt HHI: 5,535"  ✓  source: USGS MCS 2025
  "LFP cycle life: +200%"  ✓  source: NREL Battery Performance Report 2023
  "Qualification: 80 weeks"  ✓  source: IEC 62660-1, UN 38.3, UL 2580

This is what makes the output trustworthy enough to take to procurement.


SLIDE 8 — WHY THIS IS A NET-NEW AGENT
---------------------------------------
Track 1 requires a genuinely new agent — not a wrapper.

What MatRes is NOT:
- Not a supply chain dashboard (Interos, Resilinc operate at dashboard level)
- Not a data lookup tool (USGS, Materials Project have no agent layer)
- Not a general-purpose LLM query

What makes it novel:
- First agent that connects geopolitical risk → material failure data → substitution → qualification in one pipeline
- Hallucination guard enforces citation discipline — purpose-built for regulatory-grade output
- Composite scorer with tunable weights — engineers adjust priorities (cost vs. performance vs. CO₂)
- Sub-agents are deterministic data tools (FunctionTool, not LLM calls) — fast, auditable, cheap
